Distributions¶

Distribution log likelihoods for building Bayesian models.

These should all automatically sum the log likelihoods if x is a numpy array.

distributions.bernoulli(k, p)¶

Log likelihood for the bernoulli distribution.

Parameters:
  • k – int, np.array. Number of successes.
  • p – int, float, np.array. Success probability.

Special case of binomial distribution, with n set to 1.

distributions.beta(x, alpha=1, beta=1)¶

Log likelihood of beta distribution.

Parameters:
  • x – float, np.array. Must be between 0 and 1.
  • alpha – (optional) int, float. Shape parameter, must be greater than 0.
  • beta – (optional) int, float. Shape parameter, must be greater than 0.
\[\log{P(x; \alpha, \beta)} \propto (\alpha - 1)\log(x) + (\beta - 1) \log(1 - x)\]
distributions.binomial(k, n, p)¶

Log likelihood of the binomial distribution.

Parameters:
  • k – int, np.array. Number of successes.
  • n – int, np.array. Number of trials.
  • p – int, float, np.array. Success probability.
\[\log{P(k; n, p)} \propto k \log(p) + (n-k)\log(1-p)\]
distributions.exponential(x, rate=1)¶

Log likelihood of the exponential distribution.

Parameters:
  • x – int, float, np.array.
  • rate – (optional) int, float, np.array. Rate parameter, defaults to 1, must be greater than 0.
\[\log{P(x; \lambda)} \propto \log{\lambda} - \lambda x\]
distributions.normal(x, mu=0, sig=1)¶

Log likelihood of a normal distribution.

Parameters:
  • x – int, float, np.array.
  • mu – (optional) int, float, np.array. Location parameter of the normal distribution. Defaults to 0.
  • sig – (optional) int, float. Standard deviation of the normal distribution, sqrt(variance). Defaults to 1.
\[\log{P(x; \mu, \sigma)} \propto -\log{\sigma} - \frac{(x - \mu)^2}{2 \sigma^2}\]
distributions.outofbounds(*conditions)¶

Utility function for catching out of bound parameters. Returns True if any of the conditions aren’t met. Typically you’ll use this at the beginning of defining the log P(X) functions. Example

def logp(x, y):
    # Bound x and y to be greater than 0
    if outofbounds(x > 0, y > 0):
        return -np.inf
distributions.poisson(x, rate=1)¶

Log likelihood of the poisson distribution.

Parameters:
  • x – int, float, np.array. Event count.
  • rate – (optional) int, float, np.array. Rate parameter, defaults to 1, must be greater than 0.
\[\log{P(x; \lambda)} \propto x*\log{\lambda} - \lambda\]
distributions.uniform(x, lower=0, upper=1)¶

Log likelihood of uniform distribution.

Parameters:
  • x – int, float, np.array.
  • lower – (optional) int, float. Lower bound, default is 0.
  • upper – (optional) int, float. Upper bound, default is 1.
\[\log{P(x; a, b)} = -n\log(b-a)\]
